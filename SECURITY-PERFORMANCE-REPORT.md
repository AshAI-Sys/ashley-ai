**Usage Example**:
```typescript
import { cacheAside, CacheKeys, CacheTTL, CacheTags } from '@/lib/cache-utils';

const stats = await cacheAside(
  CacheKeys.dashboardStats(workspaceId),
  async () => await fetchFromDatabase(),
  {
    ttl: CacheTTL.MEDIUM,  // 5 minutes

```

**Cache TTL Strategy**:

